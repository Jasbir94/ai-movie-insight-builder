# 🎬 AI Movie Insight Builder

A premium, AI-powered movie analysis platform that deep-dives into audience sentiment, live YouTube signals, and real-time movie intelligence.

![Movie Insight Builder Logo](/public/logo.png)

## ✨ Features

- **🧠 AI Sentiment Analysis**: Deep-dive into audience reviews with Hugging Face AI (Llama-3-8B).
- **🎬 Cinematic Insights**: High-resolution movie details, posters, and cast credit profiles.
- **📊 Engagement Scoring**: Our custom AI algorithm ranks similar movies using IMDb ratings and live YouTube trailer signals (views, comments, likes).
- **🔥 Trending Mechanics**: Live viewer tickers, session streaks, and "Fire" badges to highlight the most buzz-worthy titles.
- **🎨 Premium UI**: Glassmorphism, animated transitions (Framer Motion), and a dynamic dark-mode aesthetic.

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ai-movie-insight-builder.git
   cd ai-movie-insight-builder
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Create a `.env.local` file (use `.env.example` as a template).
   - Add your API keys for TMDB, YouTube (Google Cloud), and Hugging Face.

### Running the App

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or 3001) in your browser.
